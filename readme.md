# 📐 index_root

A simple command-line utility written in C that calculates the **n-th root** of a series of numbers.

---

## 🚀 Build

Build using CMake

### 🔧 Syntax

**On Linux**
```bash
index_root <root_index> <number1> <number2> ...

```

**On Windows**
```powershell
.\index_root.exe <root_index> <number1> <number2> ...
```
